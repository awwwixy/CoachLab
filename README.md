# CoachLab

Онлайн-платформа тренера (курсова робота). Користувач реєструється, входить в кабінет, оплачує курс (симуляція), дивиться відеоуроки, виконує завдання та стежить за прогресом.

Стек: HTML, CSS, JavaScript на клієнті та Node.js + Express на сервері. Дані зберігаються у файлі `db.json`.

## Запуск

Потрібен Node.js (16+).

```bash
npm install
npm start
```

Далі відкрити http://localhost:3000

Демо-акаунт: `demo@coachlab.ua` / `1234`

## Структура

```
server.js        — сервер (Express, обробка запитів)
db.json          — дані: курс, уроки, користувачі
public/
  index.html     — головна
  login.html     — вхід / реєстрація
  dashboard.html — кабінет
  css/style.css  — стилі
  js/            — api.js, auth.js, dashboard.js
  images/        — картинки уроків
```

## Примітки

Проєкт навчальний, тому спрощений: паролі зберігаються у db.json як текст, замість повноцінної БД використовується json-файл, а оплата — симуляція без реальної платіжної системи.
